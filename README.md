## 基本使用流程

1. 選擇你的目標比賽，例如多彎賽道、少彎高速、拉力賽、越野賽、直線加速或甩尾。
2. 選擇車輛取向，例如平衡、出彎加速、高速尾速、高速穩定、入彎轉向或濕滑 / 顛簸抓地。
3. 選擇引擎曲線與驅動類型，讓工具更接近你的車輛特性。
4. 輸入車重、前配重、馬力與扭力。
5. 依照畫面產生的調校卡片，把數值填入 Forza 的調校選單。
6. 按「複製數值」保存目前設定，或進入「開始試車」依實際症狀微調。

## 可以產出什麼結果

FH6 Tune Lab 會產出一組完整的基礎調校建議，包含：

- 前後胎壓
- 終傳比
- 外傾角、前後束、主銷後傾
- 防傾桿
- 彈簧與車高
- 回彈阻尼與壓縮阻尼
- 空力
- 煞車平衡與煞車壓力
- 差速器設定

每個調校項目都會以卡片顯示，並附上簡短說明。卡片上的 `?` 可以查看更完整的調整理由。

## 齒比計算器

「齒比計算器」可以幫你估算終傳比與各檔齒比。使用時輸入：

- 檔位數
- 車輛卡顯示的理論極速
- 1-97 km/h 加速時間
- 0-161 km/h 加速時間
- 終傳比
- 齒比優化目標

你可以選擇平衡配置、起步加速、彎道回速或極速延伸。若選擇彎道回速，還可以輸入常用出彎速度與主要回速檔位，讓齒比更貼近實際賽道需求。

齒比頁面會顯示建議終傳比、最高檔目標、1 檔紅線目標、齒比跨度，以及每一檔的建議齒比。

## 試車症狀建議

試車後可以按「開始試車」，勾選你遇到的狀況。工具會根據症狀列出應該優先調整的項目。

可選症狀包含：

- 入彎推頭、彎中推頭、出彎推頭
- 車尾太滑、車尾亂甩、高速不穩
- 整體抓地不足、前輪或後輪抓地不足
- 車身一直跳、壓路肩會飛、落地不穩
- 煞車鎖死、煞車甩尾、煞車距離太長
- 起步打滑、補油後滑動、出彎無法提早補油
- 山路、爛路、草地、跳台與路面切換問題
- AWD 常見推頭與穩定性問題

建議會以「先改哪裡、改多少、要注意什麼」的方式呈現，適合拿來做下一輪試車前的小幅修正。

## 建議使用方式

- 先用工具產生基礎值，不要一開始就大幅改動。
- 每次試車只改一到兩個項目，方便判斷效果。
- 如果車輛在某個問題上變好，但另一個問題變差，回到「試車症狀」頁重新勾選目前最明顯的狀況。
- 如果遊戲裡的滑桿無法輸入完全相同的數值，使用最接近的值即可。
- 這個工具產生的是調校起點，不是唯一正解；不同車種、升級零件、胎種和駕駛習慣都會影響最終設定。
