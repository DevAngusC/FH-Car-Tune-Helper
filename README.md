# FH6 Tune Lab

FH6 Tune Lab 是一個非官方 Forza Horizon 6 車輛基礎調校輔助工具。使用者可以輸入比賽目標、賽道型態、調校取向、引擎曲線、驅動形式、車重、配重、馬力、扭力、胎規格、可調範圍與車輛基準數據，工具會依這些條件產出一組可作為試車起點的建議調校。

這個工具的定位不是直接給出唯一正確答案，而是幫玩家快速建立合理的初始數值，再依照車輛手感、賽道與個人駕駛習慣繼續微調。

## 版本狀態

目前分支版本：`v0.9.1`

此版本先以獨立分支發布，暫不同步或合併到 `main`。

## v0.9.1 新增功能

### 基準側向 G 輸入

基礎配置新增兩個車輛基準數據：

- `97 km/h 側向 G`
- `193 km/h 側向 G`

這兩個數值會被視為調校前的車輛特性，用來判斷車輛比較偏低速機械抓地、高速空力穩定，或是兩邊都接近均衡。

為了避免參數循環，這兩個欄位應該輸入車輛原始或調校前的基準數據，不建議把工具建議後又變動的結果反覆回填。

### 抓地來源偏好

新增抓地來源偏好控制：

- 自動模式：由工具依基準側向 G、賽道型態與目前配置判斷空力抓地和機械抓地比例。
- 手動模式：使用者可以用滑桿自行偏向空力抓地，或偏向機械抓地與極速保留。

這個功能主要用來處理 FH 調校中常見的取捨：高速彎穩定性與直線尾速。

### 車輛特性策略

新增車輛特性策略下拉欄：

- 短板補強：優先修正車輛弱項，例如低速抓地不足或高速穩定不足。
- 均衡配置：維持工具原本較保守、較泛用的判斷。
- 優勢放大：不強行把車修到平均，而是讓車輛原本的強項更明確。

這個選項會在側向 G 呈現明顯弱點或強項時更有感。如果資料不足、車輛接近均衡，或建議值被四捨五入與可調範圍限制吃掉，切換策略時畫面上的數值變化會比較小。

### 工具判讀

基礎配置新增工具判讀區塊，會用標籤方式顯示工具目前對車輛的理解，例如：

- 低速抓地偏弱、普通或強
- 高速 G 偏弱、普通或強
- 目前空力抓地與機械抓地比例
- 工具建議的車輛特性策略

這可以讓使用者知道工具為什麼改變空力、懸吊、齒比或其他建議值，而不是只看到數字變化。

### 齒比計算器更新

齒比計算器現在會更完整參考目前配置：

- 目標比賽
- 調校取向
- 取向強度
- 引擎曲線
- 驅動形式
- 賽道型態
- 車輛特性策略
- 建議終傳比

另外，終端上限模式已調整。舊邏輯會讓目標終端速度落在紅線 RPM；v0.9.1 改成落在紅線 RPM 的 85%。

例如紅線是 `9500 RPM`，終端上限模式會以 `8075 RPM` 作為目標終端轉速。這樣可以降低最後一到兩個檔位理論上存在，但實際試車很難跑到的問題。

## 與 v0.9 的主要差異

v0.9 主要完成核心調校流程、齒比計算器、賽道型態、調校取向強度、語言切換、Dark Mode 與 UI 排版整理。

v0.9.1 則新增一層車輛特性判讀，讓工具不只看車重、動力和驅動，也會嘗試理解車輛低速與高速抓地特性：

- 新增 `97 km/h` 與 `193 km/h` 基準側向 G。
- 新增空力抓地與機械抓地的自動或手動配置。
- 新增短板補強、均衡配置、優勢放大三種車輛特性策略。
- 新增工具判讀，讓推薦邏輯更透明。
- 齒比間距會把車輛特性策略納入考量。
- 齒比終端上限模式從貼紅線改為紅線 RPM 的 85%。

## 主要功能

- Step 1 與 Step 2 整合成同一個基礎配置流程。
- 目標比賽選擇。
- 賽道型態滑桿。
- 調校取向與取向強度。
- 引擎曲線與驅動類型推薦。
- 車重、配重、馬力、扭力、胎規格與基準側向 G 輸入。
- 胎壓單位使用 BAR。
- 彈簧與空力可調範圍輸入。
- 防傾桿固定使用 `1 - 65` 範圍。
- 回彈阻尼與壓縮阻尼固定使用 `1 - 20` 範圍。
- 齒比計算器提供終傳比與各檔齒比建議。
- 試車狀況頁面。
- 複製建議數值。
- 繁體中文與英文介面。
- Light Mode 與 Dark Mode。

## 基本使用流程

1. 選擇目標比賽。
2. 設定賽道型態、調校取向、引擎曲線與驅動形式。
3. 輸入車重、配重、馬力、扭力、胎規格與可選的基準側向 G。
4. 如需要，自行調整抓地來源偏好。
5. 選擇車輛特性策略。
6. 設定彈簧與空力可調範圍。
7. 查看工具判讀與建議調校值。
8. 進入齒比計算器產生終傳比與各檔齒比建議。
9. 開始試車，再依實際車況回來微調。

## 注意事項

- 本工具提供的是 FH 風格調校起點，不是真實賽車工程模擬器。
- 側向 G 建議填入調校前基準值，避免讓調校後結果反覆影響下一輪判斷。
- 車輛特性策略不是每次切換都一定會大幅改變數值；資料不足或車輛接近均衡時，工具會保持保守。
- 某些建議值會因為遊戲可調範圍、使用者輸入範圍或顯示四捨五入而看起來變化較小。
- 齒比、空力與懸吊建議仍需要實際試車確認。
