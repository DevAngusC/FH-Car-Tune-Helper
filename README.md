# FH6 Tune Lab

FH6 Tune Lab 是一個非官方 Forza Horizon 6 車輛基礎調校輔助工具。使用者可以輸入比賽目標、賽道型態、彎型分布、調校取向、引擎曲線、驅動形式、車重、前後配重、馬力、扭力與前後胎尺寸，工具會依這些條件產出一組可作為試車起點的基礎調校建議。

這個工具的定位不是直接給出「唯一正確答案」，而是幫玩家快速建立一組合理的初始數值，再依照車輛手感、賽道與個人駕駛習慣繼續微調。

## v0.9 更新重點

v0.9 主要改善齒比計算器與英文介面文字，讓齒比建議更容易理解，也減少最高檔終端速度的判讀混淆。

### 與 v0.8 的差異

- 齒比計算器新增「終端檔位目標」選項。
- 新增「終端上限模式」：目標終端速度會對齊最高檔紅線 RPM。
- 新增「延伸餘裕模式」：目標終端速度會落在最高馬力 RPM 到紅線 RPM 之間，保留高速延伸空間。
- 最高檔顯示改為「最高檔紅線約 xx km/h」，不再顯示成換檔速度。
- 終傳比輸入欄移到檔位數上方，齒比計算器輸入順序更直覺。
- 調校應對方案增加修改原因與副作用說明，幫助使用者理解調校邏輯。
- 車況調校頁面調整建議數值顯示，減少不必要標籤與畫面跳動。
- 可調項目中的前空力與後空力分開顯示，符合遊戲中前後空力可分別改裝的情況。
- 英文介面與症狀翻譯重新整理，讓英文模式更一致。

## 可以產出什麼

工具會依目前設定產出以下建議：

- 前後胎壓
- 終傳比
- 外傾角、前後束、主銷後傾
- 前後防傾桿
- 前後彈簧與車高
- 回彈與壓縮阻尼
- 前後空力
- 煞車平衡與煞車壓力
- 前後與中央差速器設定
- 各檔位齒比建議、安全區間與換檔速度參考

如果某些改裝在遊戲內無法調整，可以在「可調項目」中關閉。工具會標示該項目無法調整，並以其他可調項目做固定補償。

## 使用方式

1. 在基礎配置輸入比賽類型、賽道型態、彎型分布、調校取向、引擎曲線與驅動形式。
2. 輸入車重、前後配重、馬力、扭力，以及前後胎尺寸，例如 `245/35R19`。
3. 依照車輛實際改裝狀態，在「可調項目」中關閉不能調整的項目。
4. 查看基礎建議數值，將數值輸入 FH6 後開始試車。
5. 如果試車後遇到轉向不足、轉向過度、車尾不穩、煞車不穩等問題，可以進入「車況調校」選取症狀。
6. 車況調校會依目前基礎數值提供優化後的建議數值。確認後可按「應用修改」，讓該次修改成為新的目前數值。
7. 如果只想查詢症狀對應的調校方向，可以使用「調校應對方案」。這個頁面不會連動目前車輛數值。
8. 需要保存目前設定時，可使用「儲存設定」下載 JSON 檔。下次可用「載入設定」回到同一台車的調校狀態。

## 齒比計算器

齒比計算器會依以下資料計算齒比：

- 檔位數
- 紅線 RPM
- 最高馬力 RPM
- 目標終端速度
- 終端檔位目標
- 終傳比
- 前後胎尺寸
- 驅動形式
- 目前基礎配置中的比賽、賽道、彎型、引擎與調校取向

「終端檔位目標」提供兩種模式：

- 終端上限模式：目標終端速度會剛好碰到最高檔紅線 RPM，適合想讓最高檔上限貼近指定速度的設定。
- 延伸餘裕模式：目標終端速度會落在最高馬力 RPM 到紅線 RPM 之間，適合想保留高速延伸、上坡或尾流餘裕的設定。

工具會提供每個檔位的建議值、安全區間、紅線速度參考，以及偏長或偏短可能造成的影響。

## 注意事項

- 本工具提供的是基礎調校起點，不是保證最快的最終設定。
- FH6 內不同車輛、改裝套件、輪胎、動力曲線與玩家操作習慣都會影響實際手感。
- 若遊戲內某個項目無法調整，工具只能用其他可調項目做補償，無法完全取代該項目的作用。
- 齒比建議仍需要實際試車確認，特別是終端速度、空力阻力與車輛實際拉轉能力。
